HTML Templates

HTML5 allows us to make template elements. A template can best be described as a mold with a fixed layout, which can be repeated multiple times. Instead of putting the content directly in the tags (as we did in our level 000 HTML course), the content and the tags are completely separated. This practice gives much flexibility and endless possibilities.

Making a template with Handlebars.js or Polymer

Luckily there are tools available which make the use of templates a lot easier. When you use Handlebars.js or Polymer's template binding you barely have to mess around with any Javascript. Both systems have similar functions but their syntax is a little different. For instance templates "directives" in Polymer are written inside the tags as attributes, while handlebars does it outside the tags.
